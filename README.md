# 🚦 Road Accident Analysis Dashboard (2021–2022)

## 📌 Project Overview

An interactive Microsoft Excel dashboard developed to analyze and visualize road accident data for **2021 and 2022**. The dashboard converts raw accident records into meaningful insights using Pivot Tables, Pivot Charts, KPI Cards, Slicers, Timelines, Hyperlinks, and Dynamic Visualizations.

The project aims to identify accident trends, casualty severity, vehicle involvement, road surface conditions, area-wise distribution, and lighting conditions to support data-driven decision-making and road safety analysis.

---

## 🎯 Project Objectives

* Analyze total casualties reported during 2021 and 2022.
* Understand the distribution of Fatal, Serious, and Slight casualties.
* Identify monthly casualty trends and seasonal patterns.
* Compare accident occurrences in Urban and Rural areas.
* Examine the impact of road surface conditions on accidents.
* Analyze accidents under different lighting conditions.
* Determine the most frequently involved vehicle categories.
* Build an interactive dashboard for efficient data exploration.
